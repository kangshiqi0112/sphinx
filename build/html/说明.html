说明¶

使用说明

  • 在index.rst文件中引入新增加的文件

  • 执行命令：sphinx-build -b html source build

  • 执行命令：make html

  • pdf生成：make latexpdf

介绍

  • source文件夹：存放源文件（rst/txt/md）、资源文件（图片等）、配置文件（conf.py）

  • build文件夹：编译后的文件（HTML、PDF）
